RSA Шифрование на Java

Проект демонстрирует практическую реализацию асимметричного шифрования RSA на языке Java. 
Приложение генерирует ключевую пару, шифрует сообщение и расшифровывает его с сохранением всех данных в файлы.

🔐 Что такое RSA?
RSA (Rivest–Shamir–Adleman) — один из первых и наиболее распространённых алгоритмов асимметричного шифрования. Он использует два ключа:
1) Публичный ключ — для шифрования (можно передавать по открытым каналам);
2) Приватный ключ — для расшифровки (должен храниться в секрете).
   
Структура проекта:
Проект состоит из двух отдельных программ:

1. RSACipherEncrypt.java — Шифрование:

1) Генерирует пару ключей RSA (512 бит);
2) Сохраняет ключи в файлы (public.key, private.key);
3) Шифрует сообщение;
4) Сохраняет зашифрованный текст в файл (encrypted.txt).

2. RSACipherDecrypt.java — Расшифровка:

1) Загружает приватный ключ из файла;
2) Читает зашифрованное сообщение;
3) Расшифровывает и выводит исходный текст.

Технологии:
1) Java Cryptography Architecture (JCA);
2) RSA алгоритм (ключ 512 бит);
3) Base64 кодирование для текстового представления;
4) PKCS8EncodedKeySpec для работы с ключами.

📂 Файлы, создаваемые программой:
1) public.key	Публичный ключ в формате Base64;
2) private.key	Приватный ключ в формате Base64;
3) encrypted.txt	Зашифрованное сообщение в Base64.

⚠️ Важные замечания:
1) Размер ключа 512 бит используется для демонстрации. В реальных проектах рекомендуется использовать 2048 или 4096 бит;
2) RSA не подходит для шифрования больших объёмов данных — максимальный размер сообщения ограничен размером ключа;
3) Base64 кодирование используется для удобного хранения и передачи бинарных данных в текстовом виде.
